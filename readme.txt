== Description ==

ClickServ is an automation system that automatically synchronizes your WooCommerce store with your SageOne account.

It syncs orders, stock and products automatically once you've set up your syncing profile.

Currently we only support WooCommerce and SageOne, however, Xero and various other accounting platforms as well as other e-commerce platforms are part of our roadmap.

Find out more at [ClickServ](https://clickserv.co.za/ "e-commerce integration and automation")

== Installation ==

How to get started with ClickServ

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->ClickServ screen to configure the plugin, this will redirect you to our portal.
4. Create a ClickServ account and integration setup wizard.
5. Approve WooCommerce API authentication.
6. Enjoy your integration experience.
